# Практична робота 4

## Огляд
Цей проєкт реалізує командний інтерфейс для `text_system` і `game_system`, дотримуючись принципів SOLID. Інтерфейс дозволяє взаємодіяти з системами через консольні команди.

## Командний інтерфейс
- **Режими**: `--text` (для роботи з текстом), `--chars` (для роботи з персонажами).
- **Команди**:
  - Text:
    - `pwd` — виводить поточний шлях (наприклад, `/test`).
    - `print [--whole|--id]` — виводить поточний елемент або весь документ з опцією `--whole`, або елемент за `--id`.
    - `add_text <container|leaf> <type>` — додає новий елемент (наприклад, `heading` або `paragraph`).
    - `rm [<elem_name>]` — видаляє поточний елемент або за ім’ям із підтвердженням.
    - `up` — переходить до батьківського елемента.
    - `cd <path>|[--id <container_id>]` — змінює поточний шлях або контейнер за ID.
  - Characters:
    - `create <char|item|ability>` — створює об’єкт у діалоговому режимі.
    - `add_to_char --char_id <id/name> --id <item/ability_id>` — додає предмет або здібність до персонажа.
    - `act <attack|heal|ability> <actor> <target> [--id <id/name>]` — виконує дію з опцією здібності.
    - `ls [--id <id/name>]` — виводить список об’єктів або деталі за ID.

## Як запустити
1. Перейдіть до `D:\Practical 4`.
2. Запустіть: `python -m main --text` або `python -m main --chars`.
3. Увійдіть в інтерактивний режим і введіть команди (наприклад, `> add_text container heading`).
